# InovaHub - Painel de Serviços Digitais

Aplicação front-end criada com React + Vite para representar um painel moderno de serviços digitais de uma startup fictícia.

## O que o projeto atende

- Estrutura em React com Vite.
- Componentização com `Header`, `HeroSection`, `ServiceCard`, `HighlightSection` e `Footer`.
- Uso de Design System híbrido com tokens CSS, padrões de cards, botões, grid responsivo e ícones do `lucide-react`.
- Conteúdo dinâmico por props nos cards, hero e seção de destaque.
- Layout responsivo para desktop, tablet e celular.
- Estilização personalizada em `src/App.css` e tokens globais em `src/index.css`.

## Como executar

```bash
npm install
npm run dev
```

Depois acesse a URL exibida pelo Vite, normalmente:

```bash
http://localhost:5173/
```
